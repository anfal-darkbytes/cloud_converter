OPTIONS

Width

Height

Output height in pixels.

Fit

Sets the mode of resizing the image. "Max" resizes the image to fit within the width and height, but will not increase the size of the image if it is smaller than width or height. "Crop" resizes the image to fill the width and height dimensions and crops any excess image data. "Scale" enforces the image width and height by scaling.

WebP ("weppy") is a file format which stores images with both lossy and lossless compression. It was developed by Google, based on acquired technology of On2.

 PNG

WEBP Converter

PNG or Portable Network Graphic format is a graphic file format that uses lossless compression algorithm to store raster images. It uses 2 stage compression methods. It is frequently used as website images rather than printing as it supports only the RGB color model. So CMYK color images cannot be saved as PNG image.
